# goroutines_pipeline
Цель задания - написание программы, в составе которой должны быть:
1. Функция - аналог unix pipeline (функция ExecutePipeline в файле signer.go).
2. Применение функций многопоточного расчета хеш-сумм для более быстрой работы по сравнению с последовательным расчетом.

Наименование и назначение основных файлов проекта:
1. signer.go - разработанный код программы.
2. HW_readme.md - описание условий задания.
3. common.go - приложенные функции расчета хеш-сумм к услвоию задания.
4. main_test.go - приложенные тесты к условию задания.
5. extra_test.go - приложенные дополнительные тесты к условию задания.
